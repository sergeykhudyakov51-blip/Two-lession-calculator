# Полиграф Про

Учебный статический мини-сайт: приветственный лендинг и калькулятор оценки стоимости полиграфического заказа в браузере. Без сборки и бэкенда — достаточно открыть HTML-файлы.

## Структура

- `index.html` — главная страница с кратким описанием, демо-блоком (ползунок тиража) и переключением светлой / тёмной темы.
- `calculator.html` — полный калькулятор параметров заказа, итогов и детализации.
- `123.html` — прежняя копия калькулятора (можно использовать как запасную или удалить после перехода на `calculator.html`).

Тема оформления сохраняется в `localStorage` под ключом `polygraph-site-theme` и общая для главной и калькулятора.

## Как открыть

Откройте в браузере файл `index.html` (двойной щелчок или перетаскивание в окно браузера). Ссылки на калькулятор ведут на `calculator.html` в той же папке.
